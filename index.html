Welcome to My Website

I am a software developer living in southern Ohio. I first wandered into the wide world of Computer Science in High School. Back then, I just wanted to make my own games. Through High School I made a couple (very) simple Java games, which led to me getting a degree in Computer Science from Cedarville University. I currently work in C++. Yet, since QML uses Javascript, and I work mostly in the frontend, I still write Javascript professionally. I really like working on web and mobile applications, since it allows me to express some creativity. Ultimately I see programming as a way to serve other people, and so I feel most energized in my work when I can help make someone's life easier.

On this site you'll find an introduction to the skills I have to offer. This site itself is intended to be a small sampling of my work, as I have done my best to make this site mobile friendly. Thank you for visiting!
