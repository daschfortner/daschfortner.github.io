Skills

I first encountered responsive web design when I worked on a hackathon team to develop a website that required mobile support. Our team decided not to use any CSS frameworks such as Bootstrap. So, we ended up writing a fully responsive website using only PHP and CSS.

This experience equipped me well for my next work experience. A year and a half later, I worked for EduSource, a small software company that developed web applications. There I worked on two different web projects that also required support for multiple devices.

The first time I programmed in Javascript I was in High School using Khan Academy's programming lessons. Back then I still thought Java and Javascript were related. I've learned a lot since then.

When I worked for EduSource, I worked on AngularJS applications. During my time there, we decided to move to Typescript. It was also through EduSource that I learned about React, and I spent some time learning how to use it in my free time. Though I currently work in C++, I still use Javascript professionally. My team uses QML, which implements the ECMA standard.

At my current position, our project is entirely C++. Since I started working in C++, I have learned countless things about the language. I'm still learning. C++ is huge.

During my time at EduSource, I also worked in C#. While I was there I gained familiarity with the .NET framework and writing endpoints for RESTful APIs.

I used lots of different languages in various capacities. I've written Python scripts professionally, played around with Haskell and Scheme, written small games in Java, and written SQL queries.

I have never developed for Android in a professional environment. But, during my senior year at Cedarville, my senior design team worked to update an existing Android application. The old app (called StoryProducer) allowed a user to create slide show videos, with translated audio over the pictures. My senior design team updated StoryProducer to create FilmProducer. FilmProducer could not only create slide show videos, it also allowed the user to record translations and place them over videos.
